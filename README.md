# 59Music

一个基于 C++ 与 Vue 开发的本地音乐播放器，支持本地音乐播放、音乐列表管理以及随机播放功能。

---

## 项目简介

本项目是一个简单的本地音乐播放器，实现了音乐文件加载、播放控制以及播放列表管理等功能。

用户可以选择本地音乐文件进行播放，并支持随机播放模式。

---

## 功能介绍

* 本地音乐文件加载
* 音乐播放 / 暂停
* 上一首 / 下一首
* 音乐列表显示
* 随机播放
* 音乐进度显示
* 基础用户界面

## 运行方式

### 前端启动

```bash
cd frontend
npm install
npm run dev
```

### 后端启动

根据实际项目修改：

```bash
cd backend
mkdir build
cd build
cmake --build . --target 59music_server  
cd .\Debug\ 
./59music_server.exe       
```
